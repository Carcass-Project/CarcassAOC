2021-12-08 || Coniano

So, you have to make a piano in the console, that plays the basic notes C D E F G A H C2

Make it use QWERTYUI, for ease of use.

Use this website to get the frequencies:

https://pages.mtu.edu/~suits/notefreqs.html

RESTRICTIONS

You MUST make it in the console.

Use the correct order of keys and notes.

Don't copy some code, and try making it unique.

Use C# or ReCT to code this exercise, but C and Lua is also allowed, if you really want to.
